# Task Manager Project
## Project Overview

This project is a simple command-line based Task Manager created to demonstrate essential Git and GitHub workflows. 
It includes basic functionality such as adding and viewing tasks while also serving as a practical example for version control concepts like branching, 
merging, resolving conflicts, using pull requests, creating documentation, and integrating CI through GitHub Actions. The project structure is lightweight 
and easy to extend, making it ideal for learning and practicing VCS fundamentals.
